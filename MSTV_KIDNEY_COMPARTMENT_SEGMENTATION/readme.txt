This code implements a method for Renal Compartment segmentation in DCE-MRI Images. 

Video: https://www.youtube.com/watch?v=vLO6_J_aFKU

Paper: http://lbmedia.ece.ucsb.edu/resources/ref/MICCAI15.pdf
       http://www.ncbi.nlm.nih.gov/pubmed/27236222

Usage: 

       This code was implemented on Matlab (version: 8.5.0.197613 (R2015a)). 
       In Matlab Command Window, type 'start' to begin. The following steps, users can refer to the video demo as the link above.
